# Neon Snake Game

A fully responsive, retro-futuristic neon Snake game playable on both desktop and mobile devices.

## Features

- **Neon Aesthetics**: Cyberpunk-inspired visuals with glow effects and sharp styling.
- **Multiple Controls**: Support for WASD, Arrow keys, Swipe gestures, and on-screen D-pad buttons.
- **Responsive**: Perfectly sized for mobile and desktop screens to prevent overflow.
- **Dynamic Difficulty**: Snake movement speeds up gradually as you eat more food.
- **Smooth Animations**: Includes visual pops on score updates and animated pulsating food.

## Setup

1. Simply open \`index.html\` in your favorite modern web browser.
2. No local servers or additional dependencies are required.

## Files Structure

- \`index.html\` - The main markup and layout.
- \`style.css\` - Custom styling (Vanilla CSS).
- \`script.js\` - The game loop and input handling in Vanilla JS.
